#  WebRTC Voice Chat

Онлайн голосовой чат для тестирования связи с друзьями в разных городах.

##  Быстрый старт

1. Перейдите по ссылке: https://your-app.railway.app
2. Введите имя
3. Введите ID комнаты (или создайте новую)
4. Отправьте ссылку другу
5. Начните разговор!

##  Технологии

- Node.js + Express
- Socket.io для сигналинга
- WebRTC для P2P аудио
- HTML5 Audio API

##  Доступ онлайн

Приложение работает в браузере, не требует установки.

##  Поддержка

Если есть проблемы со звуком:
1. Проверьте разрешения микрофона
2. Попробуйте другой браузер
3. Нажмите " Отладка" в интерфейсе
